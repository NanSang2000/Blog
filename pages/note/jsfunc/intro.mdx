# 函数式编程范式

## 引入

函数式编程指的是一种编程范式，它把计算过程视为数学上的函数计算，并且避免使用程序状态以及易变对象。函数式编程的一个主要特点是，函数是一等公民，也就是说，函数可以作为参数传递给其他函数，也可以作为其他函数的返回值。

常见的概念有：

- 纯函数
- 高阶函数
- 柯里化

## 应用领域

React 让函数式编程发扬光大，它的核心思想是：把 UI 看作是一个函数，根据不同的状态，返回不同的 UI。这种思想在 React 中被称为“函数式编程”。

## 概念

函数式编程（Functional Programming）是一种编程范式，常见的编程范式有面向对象和面向过程。而函数式编程是将现实世界的事物和事物之间的联系抽象到程序世界。

函数式编程的特点：

- 函数是一等公民（first-class function）：函数存储在变量中，函数可以作为参数传递，函数可以作为返回值返回。
- 纯函数（pure function）：函数的返回值只依赖于它的参数，而且不会有任何副作用。
- 高阶函数（higher-order function）：函数可以接收函数作为参数，也可以将函数作为返回值返回。
